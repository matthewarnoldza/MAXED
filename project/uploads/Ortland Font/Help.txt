How to Install Fonts on Windows:  

1. Download the font file in .ttf or .otf format.  
2. If the file is in .zip format, right-click and select Extract All.  
3. To install:  
   - Right-click the font file and select Install; or  
   - Open Control Panel > Appearance and Personalization > Fonts, then drag and drop or paste the font file into the window.  
4. The font will be installed and available in your applications.  

How to Install Fonts on Mac:
1. Download the font file in .ttf or .otf format.  
2. If the file is in .zip format, double-click to extract it.  
3. Double-click the font file, then click the Install Font button in Font Book.  
4. The font will be available in all applications.
